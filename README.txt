# OstblockChess

OstblockChess is a chess game with various modes including Player vs Player, Player vs AI, and Fischer Random Chess. The game features a custom menu, sound effects, and an AI opponent with adjustable difficulty.

## System Requirements

- Python 3.6 or higher
- Pygame library
- PIL (Pillow) library
- Multiprocessing library

## Installation

1. Download the game:
    ```
    https://github.com/DaniilStreltsov/OstblockChess
    ```

2. Install the required libraries:
    ```
    pip install pygame pillow
    ```

## Launching the Game

1. Navigate to the [chess] directory:
    ```
    cd chess
    ```

2. Run the [main.py] script:
    ```
    python main.py
    ```

## Features

- **Game Modes**: Choose between Player vs Player, Player vs AI, Fischer Random Chess vs AI, and Fischer Random Chess PvP.
- **AI Difficulty**: Adjust the AI difficulty level from easy to impossible.
- **Sound Effects**: Enjoy sound effects for moves, captures, promotions, checks, and checkmates.
- **Custom Menu**: Navigate through a custom menu with animated backgrounds and custom fonts.

## Controls

- **Mouse**: Click to select and move pieces.
- **Keyboard**: 
  - `Z`: Undo the last move.
  - `R`: Restart the game.
